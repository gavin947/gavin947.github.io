MRTG流量监控安装

发表于 2016-09-04

MRTG (Multi Router Traffic Grapher)—是一个监控网络链路流量负载的工具软件，通过snmp协议得到设备的流量信息，并将流量负载以包含PNG格式的图形的HTML 文档方式显示给用户，以非常直观的形式显示流量负载。

安装平台：windows

工具原料：Windwons server 2003、VMware、ActivePerl、Mrtg、instsrv.exe、srvany.exe download

安装步骤：

  • 安装VMware虚拟机软件

  • 安装windows server 2003

    1、添加IIS、ASP、SNMP组件

    2、确认 SNMP SERVICE 服务开启，添加：只读、public

    3、IIS更改默认站点文件夹 C:\WWW\MRTG

  • 安装ActivePerl

  • 配置被监控端主机开启SNMP
  • 安装MRTG到C:\MRTG

  • 添加MRTG到系统服务

    1、把Srvany.exe 和instsry.exe放到c:\mrtg\bin目录里面，在cmd下运行命令
